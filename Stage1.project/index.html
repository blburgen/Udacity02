Stage 1: Make a Web Page

Understanding CSS
CSS stands for Cascading Style Sheets. With CSS the most specific rule is applied to every element. Cascading means that you start with the global laws and keep going down into the rules until you find the one that best describes the elements. CSS is code written to control the "style" of HTML elements. To reference your css file you put the reference in the header. See the following example of referencing a css file: 
            <head>
                <link rel="stylesheet" href="style.css">
            </head>

            Example of CSS that makes all h1 elements have a black background
            and white text

            h1 {
              background-color : black;
              color: white;
            }
CSS reference material
There are three was of including CSS styling in your page
  • Write CSS in the <head> of your HTML
  • Link your HTML to a separate CSS file
  • Write your style inline with your HTML
Why is style important
  • Write more efficient code
  • Understand code more efficiently
  • Communication in a common style
Increases productivity.
